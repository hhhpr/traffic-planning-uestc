# 项目描述
首先随机生成20个蓝噪声样本点确保点之间的距离不小于半径

然后根据点之间的距离来生成边，并限制每个点边的条数，保障可视化效果

使用Dijkstra算法找出从Node0到Node20的最短路径并初始化流量图

循环迭代：更新时间t值，找最短路径，更新流量q值，并进行可视化显示
